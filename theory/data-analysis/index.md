## 데이터 분석이란?

데이터 분석은 원시 데이터에서 의미 있는 인사이트를 추출하는 과정입니다. 통계적 방법과 프로그래밍 도구를 활용하여 데이터를 탐색하고, 패턴을 발견하며, 의사결정에 활용합니다.

---

## 주요 학습 주제

### 🔬 탐색적 데이터 분석 (EDA)
- 데이터 로딩 및 기본 탐색
- 결측값(Missing Values) 처리
- 이상값(Outlier) 탐지 및 처리
- 데이터 분포 이해

### 📊 Pandas 핵심 기능
- DataFrame 생성 및 조작
- 데이터 필터링, 정렬, 그룹화
- `groupby`, `pivot_table`, `merge`
- 시계열 데이터 처리

### 🔢 NumPy 기초
- ndarray 생성과 연산
- 브로드캐스팅(Broadcasting)
- 행렬 연산
- 통계 함수

### 📉 데이터 시각화
- Matplotlib 기본 그래프
- Seaborn 통계 시각화
- 히스토그램, 산점도, 박스플롯
- 상관관계 히트맵

### 📐 기초 통계
- 기술 통계 (평균, 중앙값, 분산)
- 상관관계 분석
- 가설 검정 기초
- 분포와 확률

## 분석 워크플로우

```
1. 문제 정의
2. 데이터 수집
3. 데이터 전처리 (정제, 변환)
4. EDA (탐색적 분석)
5. 인사이트 도출
6. 결과 시각화 및 보고
```
