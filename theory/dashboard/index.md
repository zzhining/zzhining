## 대시보드란?

데이터 대시보드는 복잡한 데이터를 시각적으로 표현하여 의사결정을 돕는 인터랙티브 인터페이스입니다. Python 생태계에서는 Streamlit, Dash, Plotly 등이 널리 사용됩니다.

---

## 주요 학습 주제

### 🚀 Streamlit
- 빠른 웹앱 프로토타이핑
- `st.sidebar`, `st.columns` 레이아웃
- 위젯 (슬라이더, 버튼, 선택박스)
- 실시간 데이터 업데이트
- `st.cache_data` 성능 최적화
- Streamlit Cloud 배포

### 📊 Plotly
- 인터랙티브 그래프 기초
- `plotly.express` 빠른 시각화
- `plotly.graph_objects` 세부 커스텀
- 애니메이션 차트
- 지도 시각화 (choropleth, scatter_mapbox)

### 🎛️ Dash
- Dash 앱 구조 이해
- 레이아웃 컴포넌트 (dcc, html)
- 콜백(Callback) 함수
- 멀티 페이지 앱

### 🎨 시각화 베스트 프랙티스
- 차트 유형 선택 가이드
- 색상 이론과 접근성
- 모바일 반응형 디자인
- 효과적인 데이터 스토리텔링

## 도구 비교

| 도구 | 장점 | 단점 | 추천 용도 |
|------|------|------|---------|
| Streamlit | 빠른 개발, 쉬운 배포 | 커스텀 제한 | 프로토타입, 내부 도구 |
| Dash | 유연한 커스텀 | 학습 곡선 | 복잡한 앱 |
| Plotly | 다양한 차트 | 단독 앱 불가 | 시각화 컴포넌트 |
