## 생성형 AI란?

생성형 AI(Generative AI)는 텍스트, 이미지, 오디오, 코드 등 새로운 콘텐츠를 생성할 수 있는 인공지능 시스템입니다. 대규모 언어 모델(LLM)을 기반으로 작동하며, 방대한 데이터로 학습된 모델이 사람과 유사한 결과물을 생성합니다.

---

## 주요 학습 주제

### 🤖 대규모 언어 모델 (LLM)
- Transformer 아키텍처 이해
- GPT, Claude, Gemini, LLaMA 모델 비교
- 토큰화(Tokenization)와 임베딩(Embedding)

### 📝 프롬프트 엔지니어링
- 효과적인 프롬프트 작성법
- Chain-of-Thought 프롬프팅
- Few-shot, Zero-shot 학습
- 역할 부여와 시스템 프롬프트

### 🔧 API 활용
- OpenAI API 사용법
- Anthropic Claude API
- 스트리밍 응답 처리
- 비용 최적화 전략

### 🛠️ 실전 활용
- RAG (Retrieval-Augmented Generation)
- Fine-tuning 기초
- LangChain, LlamaIndex 프레임워크
- AI 에이전트 구축
