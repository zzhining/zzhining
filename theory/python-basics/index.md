## 파이썬이란?

Python은 읽기 쉽고 간결한 문법으로 유명한 범용 프로그래밍 언어입니다. 데이터 과학, AI, 웹 개발, 자동화 등 다양한 분야에서 활용됩니다.

---

## 주요 학습 주제

### 📌 기본 문법
- 변수와 데이터 타입
- 조건문 (if/elif/else)
- 반복문 (for, while)
- 예외 처리 (try/except)

### 📦 자료구조
- 리스트(List), 튜플(Tuple)
- 딕셔너리(Dictionary), 집합(Set)
- 리스트 컴프리헨션
- 제너레이터(Generator)

### 🔧 함수와 모듈
- 함수 정의와 매개변수
- 람다(Lambda) 함수
- 데코레이터(Decorator)
- 모듈과 패키지 구조

### 🏛️ 객체지향 프로그래밍
- 클래스(Class)와 객체(Object)
- 상속(Inheritance)과 다형성
- 매직 메서드(Dunder Methods)
- 추상 클래스

### 🗂️ 파일 처리
- 파일 읽기/쓰기
- CSV, JSON 처리
- 경로 처리 (pathlib)

| 라이브러리 | 용도 |

| `matplotlib` | 데이터 시각화 |
| `requests` | HTTP 요청 |
| `re` | 정규 표현식 |
